# Agent Skills

Personal collection of agent skills.

## Install

```bash
# Install all skills
npx skills add 4ier/agent-skills

# Install specific skill
npx skills add 4ier/agent-skills --skill offload2

# Install globally
npx skills add 4ier/agent-skills -g

# Install to specific agent
npx skills add 4ier/agent-skills -a claude-code
```

## Skills

| Skill | Description |
|-------|-------------|
| [offload2](skills/offload2) | Cognitive pressure release valve for agent execution |
